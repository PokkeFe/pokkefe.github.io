Spotify - My Current Albums

Python Album Loader

The album slider you see above was "dynamically" created using Python and Javascript

Using the spotipy library, I queried the Spotify Web API and retrieved a few of my currently saved albums.

Python

outputfile = open("savedalbums.txt",'w')

sp = spotipy.Spotify(auth=token)

results = sp.current_user_saved_albums(limit=50)

for t in results['items']:
outputfile.write((t['album']['name']+'~'+t['album']['images'][0]['url']+'~'+t['album']['uri']+'|'))

I then took the output generated by the file, and assigned it to a variable in Javascript, which I used to convert the string back into an object.

Looping through an array of each of these objects, I created an album element for each corresponding object and loaded them into a document fragment.

Finally, I appended that document fragment to my container div, and styled it up with good old CSS!
